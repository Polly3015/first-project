# Команды для терминала

1. cd - для смены директории
2. ls - для вывода содержимого директории
3. touch -  создание файла
4. mkdir - создание директории
5. cat - не кошка, а чтение файла (текстового)
6. rm - удаление файла
7. rmdir - удаление папки
8. rm -r - удаление папки со всем содержимым
9. Хэш - идентификатор коммита
10. HEAD - указывает на последний сделаный комми
